RouteResult
Represents the results of a Route(String, ActionRouteResult) operation. An instance of this class indicates whether the operation was successful through its Success property. Successful operations will have a value for Message, and may have a value for Result, but will not have a value for Exception. Unsuccessful operations will have a value for Exception, but not for Message or Result.
