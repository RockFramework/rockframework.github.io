StringSenderMessage(String, Encoding)
Initializes a new instance of the StringSenderMessage class using Text for its format. The value of the BinaryValue property will be the result of a call to the GetBytes(String) method of encoding. If encoding is null, UTF8 is used.
StringSenderMessage(String, MessageFormat, Encoding)
Initializes a new instance of the StringSenderMessage class. If the value of messageFormat is Binary, then the value of the BinaryValue property will be the result of a call to FromBase64String(String). Otherwise, the value of the BinaryValue property will be the result of a call to the GetBytes(String) method of encoding. If encoding is null, UTF8 is used.
