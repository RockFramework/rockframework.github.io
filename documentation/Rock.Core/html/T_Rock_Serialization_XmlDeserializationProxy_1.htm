Remarks
The XmlDeserializationProxyTTarget class is flexible in the xml that it accepts. For example, we want to obtain an instance of the FooContainer class:
Copy
public class FooContainer
{
    public XmlDeserializationProxy<IFoo> Foo { get; set; }
}

public interface IFoo
{
    void FooBar();
}

public class Foo : IFoo
{
    private readonly string _bar;

    public Foo(string bar)
    {
        _bar = bar;
    }

    public void FooBar()
    {
        Console.WriteLine("Foo: {0}", _bar);
    }
}
In order to obtain an instance of the 'FooContainer' class, use standard xml serialization.
Copy
string xml = GetXml();

FooContainer fooContainer;

using (var stringReader = new StringReader(xml))
{
    var serializer = new XmlSerializer(typeof(FooContainer));
    fooContainer = (FooContainer)serializer.Deserialize(stringReader);
}
Now we can create an instance of the 'IFoo' interface, using the 'Foo' property.
Copy
IFoo foo = fooContainer.Foo.CreateInstance();
We can use this xml to deserialize an instance of FooContainer. Note that the 'Foo' element has a 'type' attribute that describes the type that should be created by the XmlDeserializationProxy<IFoo>.
Copy
<FooContainer>
  <Foo type="MyNamespace.Foo, MyAssembly">
    <Bar>abc</Bar>
  </Foo>
</FooContainer>
Note that the 'Bar' property of the 'Foo' class is specified by an element. We can also specify the 'Bar' property with an xml attribute.
Copy
<FooContainer>
  <Foo type="MyNamespace.Foo, MyAssembly" Bar="abc" />
</FooContainer>
These dynamic properties are also case-insensitive.
Copy
<FooContainer>
  <Foo type="MyNamespace.Foo, MyAssembly">
    <bar>abc</bar>
  </Foo>
</FooContainer>

<FooContainer>
  <Foo type="MyNamespace.Foo, MyAssembly" bar="abc" />
</FooContainer>
If we want to supply a default type (and omit the 'type' xml attribute), we need to create a subclass of XmlDeserializationProxyTTarget.
Copy
public class FooProxy : XmlDeserializationProxy<IFoo>
{
    public FooProxy()
        : base(typeof(Foo))
    {
    }
}

public class FooContainer
{
    public FooProxy Foo { get; set; }
}
Now our xml doesn't need to specify the 'type' xml attribute (but it still can if it needs a type other than the default type).
Copy
<FooContainer>
  <Foo Bar="abc" />
</FooContainer>
