A cross-cutting set of libraries and services providing a consistent, well-documented, and easy-to-use API that helps developers solve the problems that nearly all enterprise applications have, such as logging, DI, AOP, and metrics. It also includes testing classes, helper classes, and time-saving extension methods.

 

Features

  • Extensible / Pluggable = you're not stuck with our implementation

Rock.Core

This library contains features and functionality used by the rest of our Framework. It also contains many useful utility classes and extension methods.
